## 使用场景

通过 JVM 监控指标，分析线上 Java 应用的性能问题，指标包括：GC 次数、堆内存大小、非堆内存大小、元空间大小、直接缓冲区大小、JVM 线程数等。

- 接入 ARMS 应用监控
