# 将请求客户端分布再地图上展示

通过ip_to_province获取IP所在省份，然后根据ip_province聚合，计算每个省份的客户端数量，
选择地图展示，省份为 ip_province ，数值 pv 


```SQL
* |
select
  ip_to_province(client_ip) as ip_province,
  count(*) as pv
group by
  ip_province
order by
  pv desc
limit
  500
```
