# 求一个数的几次幂、禁止转换、获取无穷数

用 pow(2,3) 函数求 2 的 3 次幂，
用 to_base(180,2) 函数把 180 转换为二进制，
用 infinity() 函数获取正无穷数




```SQL
|
SELECT
  pow(2, 3) as "求 2 的 3 次幂",
  to_base(180, 2) as "把 180 转换为 2 进制",
  infinity() as "获取正无穷数"
```
