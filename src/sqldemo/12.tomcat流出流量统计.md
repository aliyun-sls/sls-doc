# tomcat流出流量统计

使用time_series将数据时间对齐，然后聚合时间计算 body_bytes_sent 的和，按照时间顺序展示，
展示使用折现图，x轴时间，y轴body_sent 


```SQL
* |
select
  time_series(__time__, '10s', '%H:%i:%S', '0') as time,
  sum(body_bytes_sent) as body_sent
GROUP by
  time
ORDER by
  time
LIMIT
  1000
```
