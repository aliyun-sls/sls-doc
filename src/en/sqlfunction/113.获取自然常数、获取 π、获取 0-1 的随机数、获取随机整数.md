# 获取自然常数、获取 π、获取 0-1 的随机数、获取随机整数


用 e 获取自然常数，
用 pi 获取 π,
用 rand 获取 0-1 的随机数,
用 random 获取随机整数，
用表格展示



```SQL
|
select
  e() as "获取自然常数",
  pi() as "获取 π",
  rand() as "获取 0-1 的随机数",
  random(10) as "获取 0-10 的随机整数"
```
